# Vaishnavi Botla | Portfolio Web Application

A clean, responsive, single-page portfolio application built from the ground up to showcase modern full-stack engineering proficiency, interactive user experiences, and clean component structures.

## 🛠️ Core Technologies Used
* **Frontend Core:** React.js, JavaScript (ES6+), HTML5
* **Styling & Layout:** Custom Modular Vanilla CSS (Flexbox, Grid Layouts, Keyframe Transitions)
* **Build System:** Vite (Next-Generation Frontend Tooling)
* **Hosting:** Vercel

## ✨ Key Technical Implementations
* **Performance-First Styles:** Built without bulky external UI libraries. Used optimized, native CSS styling wrappers to maintain excellent layout clarity and fast load-speed metrics.
* **Hardware-Accelerated Interactions:** Implemented sleek hover micro-animations and smooth fade-in scrolling mechanisms using raw CSS Keyframes and CSS variables.
* **Component Modularization:** Formed decoupled, reusable layout chunks (`Hero`, `Skills`, `Experience`, `Projects`, `Contact`) structured for straightforward scaling.
